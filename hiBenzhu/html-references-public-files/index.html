header.html 和 footer.html 的代码中并不是标准的html文档格式，因为没有包含html、head、body标签！

但这并不影响index.html页面中 jQuery 的 load() 函数加载。

通过 load() 函数可以将外部的两个公共文件引入，进而形成一个完整的页面。

与此同时，并不会影响到其他页面对于公共文件的引用和使用，代码维护也更加的方便。

其他的注意事项：

jQuery 的 load() 一般是加载服务器上的文件(非本地文件)，要发布网站后通过 http 或 https 协议进行访问，本地 file 协议浏览会产生跨域问题。静态 html 文件调用 load() 方法有的浏览器支持( firefox 和 chrome )，有的不支持。

如果用原生的 Ajax 加载本地文件时就不会出现错误。当然，jQuery 的 load() 方法放在服务器上通过 http 或 https 加载还是支持的。

如果经常玩前端代码的小伙伴建议在本机装个web容器方便调试！
