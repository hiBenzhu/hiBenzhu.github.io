FakeLocation 使用详解

发表于2021-05-12更新于2021-05-19字数统计1.4k阅读时长7分阅读次数

被校园跑“囚禁”的大学生们的福音来啦！

我前几个月发现了一个贼好用的模拟跑步路线的软件🙃FakeLocation，博客访问量不大就写一写用法吧，用的人太多万一被官方封了就尴尬了，反正看到的人不多。

测试地点：浙江工业职业技术学院镜湖校区

测试机型：OPPO Find X

测试需要模拟的软件：运动世界校园版

下面是详细的安装及使用方法↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓

一、第一步把这个神奇的软件装进你的安卓手机(iPhone是不行的喔 !)

下载安装地址：官网包更新点这里，如果下的慢的话你可以点这里试试！这是软件作者大哥的weibo。

二、配置 FakeLocation

下载完成后打开软件，相信这个操作有手就行对吧

打开之后会是下面这个样子

接下来你需要点击左上角弹出列表，登录账号

输入邮箱点击下一步，没有注册过的话会弹出注册框让你注册的

这个软件不是免费滴，除了位置模拟好像是免费的其他功能都是要开通专业版的，一个月收费9块钱 (看了我教程自己操作还是挺划算的，总比别的代跑划算) 先买它一个月，够你跑的了

接下来点击列表中的路线模拟

点击图中的路线模拟区域跳转到地图界面可以自己画跑步的路线，请给足FakeLocation所有权限，不然的话可能会出问题，记住一定要给权限，起码位置的权限得给吧😂

路线模拟的步频模拟部分是要手机进入root模式才能模拟成功的，这个软件里面有很多功能都是需要手机进入root模式才能正常使用，还好我们的路线模拟不需要（不然我也不会分享出来了，root麻烦的要死），这里可以设置跑步速度，一般正常跑2公里设置5.0步-6.0步/s就可以了，是正常人跑步速度，8.0步/s是极限，不畏惧运动校园检测的话随意设置，因为步频模拟要root模式下才能模拟成功，所以各位同志们麻烦用你们智慧的双手辛苦点摇手机吧，只能物理提升步频了

因为现在是刚开始使用软件，所以可以先随便画一段路线测试

这里我画了一大圈，哈哈哈哈哈哈就是玩

点击完成之后会跳转回路线模拟界面，点击启动模拟。

没使用过FakeLocation的同学们需要前往设置开启开发者选项、关闭电池优化、保持后台运行等等，放心接下来我会讲的。

点击启动模拟后会弹出这么个玩意，先切换到NOROOT模式

然后会弹出没有开启开发者选项的提示框，点击前往设置（如果不行的话请自行开启，开启办法是打开手机设置—关于手机，点击7-8次版本号，会开启开发者选项）

打开开发者选项，提示的会崩溃啥的都是唬人的，按我说的来就不会出事！

在开发者选项中往下滑一小段距离就是 模拟位置信息应用，点击选择FakeLocation

接下来返回 FakeLocation 再次点击启动模拟，会提示允许应用后台运行

打开 FakeLocation 框框中的开关

后面提示的啥，全都开启就完事了，不会的来邮箱屌我 hi@benzhu.me

到这里就都OK了，点击停止模拟吧，接下来要干正事了！

三、根据点位画路线进行跑步

打开运动世界校园版，点击计分跑（其他的什么一起跑、自由跑啥的都能用的）

根据系统给出的点位，用 FakeLocation 进行画路线。可以将之前配置的时候画的路线删除，长按就可以了

画完路线之后点击启动模拟，然后切换到运动校园APP，滑动开始跑步

在跑步的过程中记住一定要甩哈，不然步频都飙红了！！

跑步完毕先点击运动校园APP的暂停，切换回 FakeLocation 停止模拟，再切换回运动校园APP长按结束，结束跑步

点击结束运动

跑步成功，恭喜你 get 一个新技能

四、注意事项

华为手机可能遇到跑到一半不动的现象解决办法：

在运动校园APP界面跑步的时候每隔10秒左右切换到 FakeLocation ，再切换回运动校园，如此重复直至跑步结束

模拟软件在运行但运动校园没有更新位置解决办法：

先看看运动校园左上角是不是没有信号了，这是这个软件经常会遇到的问题，目前的现状就是有些手机实施根据模拟软件更新位置，有些手机更新位置慢，但肯定是会更新位置的，重启手机试试不知道行不行，如果打不到点的话在那个点上多画线，多画几个来回就能打上点了

遇到其他问题扣邮箱：hi@benzhu.me
